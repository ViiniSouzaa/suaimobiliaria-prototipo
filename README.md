# Protótipo "Sua Imobiliária" (React + Vite)

Protótipo frontend em React + Vite — tema escuro, animações simples e várias páginas (Home, Imóveis, Detalhe, Sobre, Contato).

## Como rodar localmente

1. `npm install`
2. `npm run dev`
3. Abra `http://localhost:5173`

## Como buildar e publicar no GitHub Pages

1. Atualize `package.json` com o campo `homepage`: `"https://<SEU_USUARIO>.github.io/suailmobiliaria-prototipo"`
2. `npm run build`
3. `npm run deploy` (o script usa `gh-pages` para enviar a pasta `dist`)

Substitua `<SEU_USUARIO>` pelo seu usuário GitHub antes de publicar.
